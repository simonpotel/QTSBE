# Auto Fetch

---

This tool aims to update automatically a list of tokens.

## Create config.json

```json
{
    'Yahoo'{
        ['AMZN', '1d'],
        ['TSLA', '1d'],
        ['NKE', '1d']
    },
    'Binance'{
        ['BTC/USDT', '1d'],
        ['ETH/USDT', '1d'],
        ['SOL/USDT', '1d']
    }
}
```
